프로그램작성 규칙
1. 파일을 저장한 위치를 지정
2. 파일명 첫글자는 대문자
3. 확장자는 .java
4. 기본 인코딩은 UTF-8 
5. 클래스를 생성해야 하며 클래스 내부에 메인메서드를 만들어야 함
6. 클래스 이름은 파일이름과 **반드시 동일해야함**
7. 메인메서드 명은 -> main(String[] args)
8. 대소문자는 반드시 구분
9. 오타주의 / 글자누락 주의 / 철자법주의/통편집주의
10. 구분자 반드시 사용함 ; <- 세미콜론
11. 중,소괄호의 순서와 짝이 맞아야함 { ( ) }  /  ( { } ) 열림과 닫힘 순서를 잘 보기!!
